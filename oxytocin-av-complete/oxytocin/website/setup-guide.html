🛡️ Oxytocin AV Setup Guide

Follow these steps to deploy your website, set up PayPal payments, and publish your app. No coding experience required.

Step 1 — Upload to GitHub 5 min

1. Go to github.com and create a free account if you don't have one.

2. Click New repository. Name it oxytocin-av. Set it to Public. Click Create repository.

3. Upload all your files by clicking uploading an existing file on the repo page. Upload everything from the zip you downloaded.

4. Click Commit changes.

✓ Your code is now on GitHub!

Step 2 — Deploy website on Netlify 3 min

1. Go to netlify.com → Sign up free with GitHub.

2. Click Add new site → Import an existing project.

3. Choose GitHub → Select your oxytocin-av repo.

4. Set Publish directory to: website

5. Click Deploy site.

✓ Your website is live at something.netlify.app — copy your URL!
⚠ Want a custom domain like oxytocinav.com? Buy one on Namecheap (~$10/yr) then add it in Netlify → Domain settings.

Step 3 — Set up PayPal payments 10 min

A. Get your PayPal Client ID:

1. Go to developer.paypal.com → Log in with your PayPal account.

2. Click Apps & Credentials → Create App.

3. Name it OxytocinAV → Click Create.

4. Copy your Client ID.

B. Create subscription plans:

1. Go to paypal.com/billing/plans → Create Plan.

2. Create Pro plan: $4.99/month, name "Oxytocin AV Pro".

3. Create Business plan: $9.99/month, name "Oxytocin AV Business".

4. Copy both Plan IDs (they look like P-12ABC345...).

C. Update your website file:

Open website/index.html and replace these 3 placeholders:

YOUR_CLIENT_ID_HERE   → paste your PayPal Client ID
plan_PRO_ID_HERE      → paste your Pro plan ID  
plan_BIZ_ID_HERE      → paste your Business plan ID

Save the file and re-upload to GitHub. Netlify will auto-redeploy in ~30 seconds.

✓ PayPal payment buttons are now live on your site!

Step 4 — Publish app for download 15 min

The easiest way to host your app downloads is GitHub Releases.

1. On your GitHub repo, click Releases → Draft a new release.

2. Tag it v1.0, title it Oxytocin AV v1.0.

3. Drag and drop your installer files (the .exe, .dmg, .deb) into the release.

4. Click Publish release.

5. Copy each file's download URL (right-click → Copy link).

6. Open website/index.html, find the downloadApp function, uncomment the last line and paste your URLs:

windows: 'https://github.com/YOUR_USER/oxytocin-av/releases/download/v1.0/OxytocinAV-Setup-1.0-win64.exe'
mac:     'https://github.com/YOUR_USER/oxytocin-av/releases/download/v1.0/OxytocinAV-1.0.dmg'
linux:   'https://github.com/YOUR_USER/oxytocin-av/releases/download/v1.0/oxytocin-av_1.0_amd64.deb'
✓ Download buttons on your site now serve real files!

Step 5 — Build installers Optional

To create a real .exe / .dmg / .deb from the Python app:

# Install PyInstaller
pip install pyinstaller

# Windows — creates dist/OxytocinAV.exe
pyinstaller --onefile --name=OxytocinAV app/oxytocin_av.py

# macOS — creates dist/OxytocinAV.app
pyinstaller --onefile --windowed --name=OxytocinAV app/oxytocin_av.py

# Linux — creates dist/oxytocin-av
pyinstaller --onefile --name=oxytocin-av app/oxytocin_av.py
⚠ Run PyInstaller on each OS to build that OS's installer. You can't build a Windows .exe on macOS.

Step 6 — Install via pip Easiest

Users can install instantly with pip (no installer needed):

pip install oxytocin-av

To publish to PyPI so this works:

pip install twine build
cd app/
python -m build
twine upload dist/*

You'll need a free account at pypi.org.

✓ After publishing, anyone in the world can install with one command!

🎉 You're all set!

Your antivirus website is live, payments work, and users can download your app.
